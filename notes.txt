Different terms for inputs:
GOTCHA concept!
It means you must use different variables for different input sizes in a complexity formula when theyâ€™re independent.

Example: 
def myfunc(a,b):
    for i in range(a):
        print(i)
    for j in range(b):
        print(j)
Cannot simplify to O(n), beacause we have 2 inputs that could = n.
So instead of forcing everything into one n, you write things like O(n + m) or O(V + E) to show which part of the input each term depends on.

Greater concept:
This is a simplified way of saying that each part of a program, inside and outside of a function should be evaluated separated, and then combined to get the runtime (depending on the the execution path).
